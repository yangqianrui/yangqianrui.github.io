欧拉函数是一个积性函数，即若 gcd⁡(a,b)=1\gcd(a,b)=1gcd(a,b)=1，则 φ(ab)=φ(a)φ(b)\varphi(ab)=\varphi(a)\varphi(b)φ(ab)=φ(a)φ(b).

筛法

考虑一个问题：快速求不大于 nnn 的所有质数。

一个简单的筛法叫 Eratosthenes 筛法，是从小到大遍历每个数，对于每个找到的质数，把它所有不大于 nnn 的倍数它标为合数。则若一个数被遍历到而没有被标记，则这个数是质数。这个筛法的时间复杂度是 O(nlog⁡log⁡n)O(n\log\log n)O(nloglogn).

int Eratosthenes(int n) {
    int p = 0;
    for (int i = 0; i <= n; ++i) is_prime[i] = 1;
    is_prime[0] = is_prime[1] = 0;
    for (int i = 2; i <= n; ++i) {
        if (is_prime[i]) {
            prime[p++] = i;
            for (int j = i * i; j <= n; j += i) is_prime[j] = 0;
        }
    }
    return p;
}

一个时间复杂度更优秀的筛法叫欧拉筛（线性筛）。具体来说，还是从小到大遍历每个数，且记录每个数的最小质因子 vvv，假设已经知道第 iii 个数的最小质因子 v[i]v[i]v[i]，则枚举所有小于等于 iii 的质数 ppp，则 v[i×p]=pv[i\times p]=pv[i×p]=p. 若枚举到 iii 时 f[i]f[i]f[i] 还未求出，则 iii 是质数。显然，这个筛法中每个数的 vvv 值只会被自己的最小质因子更新一次，复杂度 O(n)O(n)O(n).

for (int i = 2; i <= n; ++i) {
    if (v[i] == 0) prime[++tot] = v[i] = i;
    for (int j = 1; j <= tot; ++j) {
        if (prime[j] > v[i] || prime[j] > n / i) break;
        v[i * prime[j]] = prime[j];
    }
}

现在考虑如何用筛法求出欧拉函数。首先 Eratosthenes 筛法求欧拉函数是显然的，每个数都会被自己的所有质因子筛一遍，则在被质因子筛到时利用欧拉函数的计算公式即可。

那如何用线性筛求欧拉函数呢？先考虑欧拉函数的两个性质：

  • 若质数 p∣np\mid np∣n 且 p2∣np^2\mid np2∣n，则 φ(n)=φ(np)×p\varphi(n)=\varphi(\frac n p)\times pφ(n)=φ(pn​)×p.

    该性质可用求欧拉函数的公式证明。

  • 若质数 p∣np\mid np∣n 但 p2∣np^2\not\mid np2​∣n，则 φ(n)=φ(np)×(p−1)\varphi(n)=\varphi(\frac n p)\times (p-1)φ(n)=φ(pn​)×(p−1).

    由于欧拉函数是积性函数，且 gcd⁡(np,p)=1\gcd(\frac n p,p)=1gcd(pn​,p)=1，故 φ(n)=φ(np)×φ(p)=φ(np)×(p−1)\varphi(n)=\varphi(\frac n p)\times \varphi(p)=\varphi(\frac n p)\times (p-1)φ(n)=φ(pn​)×φ(p)=φ(pn​)×(p−1).

有了这两条性质后，我们就能从 nnn 的最小质因子的欧拉函数推出 nnn 的欧拉函数了。

phi[1] = 1;
for (int i = 2; i <= n; ++i) {
    if (v[i] == 0) prime[++tot] = v[i] = i, phi[i] = i - 1;
